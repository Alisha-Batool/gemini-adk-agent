# Gemini AI Agent (Google ADK)

This project implements a local AI agent using Google's Agent Development Kit (ADK) and Gemini 2.5.

## Features
- Gemini 2.5 LLM integration
- Python 3.11 virtual environment
- ADK agent orchestration
- Local web interface

## Run

```bash
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install google-adk
adk web
